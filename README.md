# Happenings — Wedding Decor, Venues & Catering

Premium wedding planning platform for **Happenings** (Karachi) — browse decoration packages, compare venues, add catering, and get instant budget estimates via WhatsApp.

Built by [Fixels Media](https://fixelsmedia.com)

## Features

- 6 decoration packages with detail modals
- Venue directory with capacity & pricing
- Catering calculator with live cost updates
- Interactive budget estimator with WhatsApp integration
- Filterable photo gallery
- Instant quote form
- Sticky WhatsApp button on every screen
- Mobile-first responsive design
- Parallax hero + scroll progress bar

## Deploy

Static site — deploy to Vercel, Netlify, or any static host.

```bash
# Vercel
vercel --prod

# Or just drag the /public folder to Netlify
```

## Stack

Pure HTML/CSS/JS — no build step, no dependencies.
